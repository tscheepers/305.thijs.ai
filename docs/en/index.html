We’ve been working with a wide variety of clients since 2010. Together we create and improve software products. In our client relations, we act as genuine technical and strategic partner for the long term.

We produce modern, complex, scalable software products that are easy to use. These software products may be entirely standalone or act as an enrichment for a physical product. Such as an enriching mobile app, web platform or SaaS product.

We create products together with our clients. We work continuously in an agile product development process, with the client taking the role of product owner. We devote extra attention to creating a great user experience and keep a constant focus on ensuring products are user-friendly.

Developing
software
together

Got an idea for a new software product? Or looking to enrich an existing product with software? We’d be pleased to put our expertise to work to help you.

The core of modern software products is online. We specialise in creating web applications using Laravel and node.js We design advanced web-interfaces for all screen formats, often using React. External developers can link with your software product via a web service and API. We set up scalable, privacy-conscious application infra­structure with modern cloud providers.

When developing apps for Android and iOS we believe in a native approach. This means that we always work primarily with the newest tools supplied by Google and Apple themselves. Sometimes we have apps link up with physical products—for example by collecting sensor data via USB, Bluetooth, Wi-Fi or an IoT network. Besides apps for smart­phones, we also develop apps for desktops, tables, smart watches and televisions.

When creating software products, we always put the user’s needs first. We devote extra attention to the user experience. Not just by creating beautiful visual interfaces, but also by thinking carefully about the user’s needs when preparing technical designs and by developing smart features that anticipate user actions.

Technical and
strategic
advice

If you have an existing software product we would be pleased to advise you on how to improve your development process and help you put this into practice, ensuring your business is properly prepared to take the next big step.

Does your software need to be made more stable and scalable? Perhaps you want to develop an attractive, efficient user experience? Or to add a unique but complex feature that will set your product apart from the rest? We can help you achieve this.

Besides development and advice, we can also offer support with setting up your own internal product development team. For example, by carrying out a development transition process.

Your product belongs to you, now and in future. Whether it’s a new product or an existing one, the intellectual property rights always transfer to you automatically. In implementing this, we make provision for our dedication to open source.
